Understanding Arrays in C++

The Foundation of Data Structures

Author

Archana R. Shah Lecturer, Computer Engineering Department

  • ✨ Introduction

    In the world of programming, data structures play a vital role in organizing and storing data efficiently. One of the simplest and most commonly used data structures is the array. In this blog, we’ll explore arrays in C++, understand their structure, operations, and see them in action with examples.

    🔍 What is an Array?

    An array is a collection of elements stored in contiguous memory locations. All elements in an array are of the same data type and can be accessed using an index.

    🛠️ Declaring and Initializing Arrays

    To use an array, you must first declare it.

    Syntax:

    data_type array_name[array_size];

Example:
    int numbers[5]; // Declares an array of 5 integers

    You can also initialize it during declaration:

    int numbers[5] = {10, 20, 30, 40, 50};

    🔁 Accessing and Traversing Arrays

    You can access array elements using their index (starting from 0):

    cout << numbers[2]; // Outputs: 30

    To print all elements, use a loop:

    for (int i = 0; i < 5; i++) {
        cout << numbers[i] << " ";
    }

    Output:

    10 20 30 40 50

    🧪 Real-World Example: Average of 10 Numbers

    Let’s write a program that takes 10 integers from the user and calculates their average.

    #include <iostream>
    using namespace std;

    int main() {
        int arr[10];
        int sum = 0;

        cout << "Enter 10 numbers:" << endl;
        for (int i = 0; i < 10; i++) {
            cin >> arr[i];
            sum += arr[i];
        }

        float average = sum / 10.0;
        cout << "Average = " << average << endl;

        return 0;
    }

    ⚠️ Common Pitfalls

    • Index Out of Bounds: Accessing elements beyond the array size will lead to unexpected results.
    • Uninitialized Values: Declaring without initializing may leave garbage values.
    • Static Size: Arrays in C++ have a fixed size unless you’re using dynamic memory.
